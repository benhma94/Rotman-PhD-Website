---
title: "Connect to a Rotman Printer"
category: "Admin"
description: "How to add a Rotman printer from a Rotman-managed device."
pinned: false
---

Use this from a Rotman-managed device.

1. Open the Start Menu.
2. Select **Run**.
3. Enter `//rprint`.
4. Select the applicable printer from the list.

This printer path is only accessible from Rotman devices.
